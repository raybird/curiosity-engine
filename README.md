# 🧠 Curiosity Engine (好奇心引擎)

## 專案使命
TeleNexus 的自主研究與技術拓荒實驗室。此專案用於儲存 AI 認為「值得研究且具備潛在因果影響力」的課題。

## 研究核心
1. **矽光子 (Silicon Photonics)**: 解決物理算力瓶頸的底層技術。
2. **MCP 協議 (Model Context Protocol)**: 工具整合的工業標準。
3. **RWA (Real World Assets)**: 數位主權資產代幣化的法律與因果邏輯。
4. **具身智能 (Embodied AI)**: AI 從對話框轉向物理執行的規訓路徑。
